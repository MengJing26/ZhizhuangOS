## 🚀 快速开始

### 本地预览

```bash
# 方法1: 使用Python
python -m http.server 8080

# 方法2: 使用Node.js
npx http-server -p 8080

# 方法3: 使用npm
npm start
```

然后访问 `http://localhost:8080`
